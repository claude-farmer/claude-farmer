# 🌱 Claude Farmer — VSCode Extension

**Your code grows a farm.**

An idle pixel-art farming game that lives in your VSCode sidebar. Just code — your farm grows automatically!

## Features

- 🌱 **Auto Growth** — Your coding activity automatically plants and grows crops
- 🎨 **Pixel Art** — Cute 16×16 pixel art farm with time-based backgrounds
- 🎲 **Gacha Collection** — Harvest crops to get gacha items (24 collectibles)
- 💧 **Social** — Visit other developers' farms and water their crops

## How it works

1. Install the extension
2. Code as usual in VSCode
3. Watch your farm grow in the sidebar!
4. Crops go through 4 stages: seed → sprout → growing → harvestable
5. Harvesting triggers gacha drops with 4 rarity tiers

## Screenshots

Your pixel-art farm appears in the activity bar sidebar, with real-time growth as you code.

## Requirements

- VSCode 1.85+
- Optional: `npm install -g claude-farmer` for CLI features
